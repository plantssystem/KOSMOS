# 🎹 KOSMOS MIDI RANDOMIZER for Renoise
**KOSMOS MIDI RANDOMIZER** is a Renoise Lua tool that transforms incoming MIDI notes from KOSMOS into musically meaningful randomized output.

It supports multiple musical scales—including traditional Japanese modes such as *Hirajoshi* and *In Scale*—and remaps incoming notes in real time.  
This adds an additional layer of musical “variation” and “controlled randomness” on top of KOSMOS’s generative phrases.

---

## ✨ Features

- **Real-time MIDI Randomization**  
  Instantly transforms notes coming from KOSMOS or any MIDI keyboard.

- **Japanese Scales Included (Hirajoshi, In Scale, etc.)**  
  Generates natural, expressive melodies with unique tonal character.

- **Musical Randomness Control**  
  Adjust pitch, octave range, velocity, probability, and more via a simple GUI.

- **Native Renoise Performance**  
  Lightweight Lua implementation with fast, low-latency processing.

- **Perfect Companion for KOSMOS**  
  Hardware generation × Renoise randomization × DAW effects  
  → A powerful three-layer generative music engine.
